# Contributing

Thanks for your interest! This is a small project, so here are the basics.

## Getting Started

```bash
git clone https://github.com/tbk42/LlamaDeck.git
cd LlamaDeck
pip install -r requirements.txt
python -m backend.main
```

## Before Submitting

- Test your change against a local Ollama instance
- Update `CHANGELOG.md` under `[Unreleased]` with your change
- If it changes anything user-facing, update `README.md` too

## Code Style

- No comments in code unless absolutely necessary
- Follow the existing style — vanilla JS, dark-theme HTML, Python with type hints
- Keep it simple

## Questions

Open an issue or discussion on GitHub.
